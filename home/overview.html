JaCH: Javascript, CSS, HTML

$ROOT

$ROOT is a shorthand for the directory where the current project's files reside on my machine.

Directory Layout

This is an overview of one way to set up a collection of Javascript/CSS/HTML projects. Oftentimes, the recommendation is to put all the Javascript files in one directory (often called 'js'), all the CSS files in another directory ('css' or 'styles' or the like), and all the HTML files in yet another directory ('templates' or 'layouts' or some such).

Setting things up this way forces the developer to constantly be jumping from directory to directory and can often make things more difficult to find.

I'm trying a different structure, putting all the files related to a particular project together in one directory. So, for example, the source for this file, the front page of this site, the CSS to format them, and the Javascript for reporting each HTML file's last update (see the bottom right corner of this page) all live in the same directory, $ROOT/home/.

Each subproject (Calories Per Pound, Conway's Life, etc. -- see the front page) will get its own appropriately named subdirectory which will hold all the HTML, CSS, and JS files for that project.

Last Update Timestamps

Emacs, my favorite editor, has a feature where it will insert the current timestamp into the file being edited if a line near the top has the form 

   Updated: <>
or 
   Updated: ""
The timestamp is inserted when the file is saved, so it provides a very precise way of tracking when the file was last updated.

To exploit this, I have the following at the top of each of my HTML files: 

   <script id="updated">Updated: "TIMESTAMP"</script>

In each of my CSS files: 

   :root {
       --updated: "TIMESTAMP";
   }

And in each of my JS files: 

   var js_update = "Updated: TIMESTMP";

Finally, I have written a JS function called show_update() and arranged for it to be called when the window is loaded. 

   window.onload = show_update;

If everything is set up properly, when one of my pages is loaded, in the right lower corner will appear as display of the dates and times of the most recent updates to the HTML, the CSS, and the JS file.
