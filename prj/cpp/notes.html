CPP Notes

  • The script element at the top of the HTML file with id 'updated' documents the most recent update time of the HTML file. 

          <script id="updated">Updated: "2022.0217 19:04:22"</script>

    It is used to display the last update timestamps at the bottom right of each page. The code that handles this is function show_update() in $ROOT/home/index.js.

    To get a CSS last update timestamp, you need a the following somewhere in your HTML file's <head> section: 

          <div class="cssmeta" hidden></div>

    For the Javascript update timestamp, you need 

          var js_update = "Updated: <2022.0217 12:24:34>"

    in your Javascript file, and you must import $ROOT/home/index.js to get the show_update() function.

    The show_update() function loads the last update information into a <div> element with id 'last_update'. If that element does not exist, the show_update() function creates it and appends it to the end of the HTML <body>.
